🏪 POS System - Point of Sale Application
A modern Point of Sale (POS) system built with Flutter and Firebase, featuring inventory management, real-time sales tracking, barcode/QR scanning, voice input, and role-based access control.

📋 Features
🛒 Point of Sale
Multiple Input Methods: QR Code Scanner, Barcode Scanner, Voice Input, Manual Search

Cart Management: Add/Remove items, Update quantities, Swipe to delete

Checkout System: Multiple payment methods (Cash, Card, Mobile Payment, Credit)

Real-time Calculations: Total amount, Profit tracking

Receipt Generation: Auto-generated receipts with print option

📦 Inventory Management
Real-time product tracking with Firestore

Low stock alerts with customizable threshold

Product CRUD operations (Add, Edit, Delete)

Search and filter functionality

Stock statistics dashboard

Product categorization

👥 User Management (Owner Only)
Role-based access control (Owner, Manager, Worker)

User activation/deactivation

Role assignment and management

User profile management

⚙️ Settings
Theme: Light, Dark, System default

Currency: Multiple currency support (INR, USD, EUR, GBP, etc.)

POS Settings: Toggle profit display, Auto print receipt

Inventory: Customizable low stock threshold

Notifications: Sound effects, Vibration feedback

Data & Sync: Offline mode support, Auto sync

📊 Reports
Sales history with transaction details

Daily sales summary

Profit tracking per transaction

Payment method analytics

🏗️ Tech Stack
Frontend: Flutter (Dart)

Backend: Firebase (Firestore, Auth)

State Management: Provider

Local Storage: SharedPreferences

Key Libraries: mobile_scanner, speech_to_text, intl, email_validator

🚀 Quick Start
Prerequisites
Flutter SDK (>=3.0.0)

Firebase Account

Android Studio / VS Code
